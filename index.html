Pre-Final - 期末推測設計背景資訊_20210106

甚麼是實體？甚麼又是想像？ 實體#1泛指實際存在的物，而想像#2則為人對於某物或某事之心靈構思。 大多人對於想像都是偏向於可能或不可能，而非必然， 但想像也如記憶卡一樣也是有資訊的存儲空間， 現代已有許多虛體的事轉化為實體的物， 過程為人腦>手>電腦>建模>虛體模型匯出>匯入物件製造>產生物件， 那為何無法直接透過人為的想像直接產生物件？ 或於虛擬空間看到想得到之物實際抽取於實際空間中。

W07 - 推想練習-修訂版作業_20201104

對於架空歷史(地理/事件/...)，請參考歷史事件或地理形貌的描繪方式，除了文本故事之外，應該有基本資訊、歷史/地理/事件脈絡關係、類比與比較、...等說明與圖表。 對於星球生物的描繪，請參考各類生物誌的格式，基本上包括各類屬性、生活習性，描繪時應包括個體全貌、比例尺、頭部或重要部位的局部細節、不同成長階段的樣貌變化、群體生活關係、...等。 請留意：對一個創作者來說，推想練習的過程和結果都很重要（設計都一樣）：過程著重脈絡、根據、分析、推論過程，以及如何掌握合理與想像的比重。 結果著重概念整合、內容、表現方式，以及對他人的啟發性。


W06 - 推想練習_20201028

推想一詞表示推測與想像。挑選下述一個題目來做推想練習：立足於某個現實（過去或現在）的時空或故事背景，選擇一個時間或歷史的分歧點，建立一個架空歷史（Alternative History）的環境，試著以文字描述或圖像描繪之。 挑選一個地球之外的行星（太陽系或虛構星系），以盡可能符合科學的條件下，設想一種可以在該行星上生存的生物樣貌、以及其生存方式。以文字和圖像描述。 繳交內容：不限媒體形式與製作軟體。若有多個檔案請壓縮打包再上傳。本練習著重於推測的過程，包括如何分析時空與環境脈絡、推想的邏輯、條件與合理性的取捨等。

W03_Hypertext超文本練習(一)_20201007

運用課程中教的基本 HTML 標注語言，以純文字方式編寫，將上次作業整理的科學史分支脈絡或電影解析，轉化為超文本格式， 也就是要具備超連結、圖片、文字格式等變化。編寫完成後，資料夾中應該至少有兩個以上的 HTML 檔、以及一個以上的影像檔 （jpg, png, gif），將該資料夾打包為 zip 或 rar 等壓縮檔格式，以學號命名。 鼓勵自學更多 HTML 語法，可參考網路上更多的 HTML 基本教學，但請先不要用到較複雜的 CSS 和 JavaScript。
